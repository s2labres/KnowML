# KnowML

**KnowML: Improving Generalization of ML-NIDS with Attack Knowledge Graphs**

*IEEE Euro S&amp;P paper*

## Code Availability

The source code for this project will be made publicly available by **June 2026**.

Please check back later or watch this repository for updates.

## Abstract

KnowML is a framework that leverages attack knowledge graphs to improve the generalization of machine learning-based network intrusion detection systems (ML-NIDS).
